# モジュールの概要

現在および計画中のモジュールは、クライアントが何に焦点を当てているかを理解できるようにグループ化されています。多くのモジュールは意図的に最小限に抑えられており、軽量性を維持するために必要なコアオプションのみを備えています。

## 建築とブロック

- **EasyBridge** — スプリント中に自動的にブロックを前方に配置し、橋を建設し続けます。範囲調整可能。
- **FastPlace / FastBreak** — シンプルなスライダーで配置遅延を減らし、採掘速度を向上させます。
- **SmoothStackUp** — きれいなタワー建築のために、あなたの下にブロックを配置します。
- **AutoBuild / AutoFill** — 設計図から建築するか、マークされた領域を選択したブロックで埋めます。
- **FreeBuild** — クリエイティブスタイルの建築のために配置制限を緩和します。

## 移動

- **AirFlyer** — 速度制御付きのエリトラおよびバケツアシスト飛行。
- **AntiFall** — 水やその他の軽減策で落下ダメージを防ぎます。
- **AnyWaySprint / AutoSprint / AutoSneak** — どの方向にもスプリントを維持し、ブロックの端で自動的にスニークします。
- **AutoSwitch / AutoTool** — ターゲットに応じて適切なツールや武器に切り替えます。
- **FastMove** — 基本的な移動速度を向上させます。

## 戦闘

- **ArmorManage** — 利用可能な最高の防具を自動的に装備します。
- **AutoBlock / AutoShield** — 脅威にさらされたときにブロックまたはシールドを構えます。
- **AutoCritical** — ジャンプのタイミングを計り、より頻繁にクリティカルを発生させます。
- **AutoTotem** — 体力が閾値を下回ったときにトーテムに切り替えます。
- **AutoLeave** — 安全ルールがトリガーされたときに切断します（例：低体力時）。
- **Combat assists** — 近接または遠距離サポートのためのInFightAssist/OutFightAssist、およびConvenientHotBarを介したホットバーの利便性。

## 視認性と認識

- **FullBright** — 世界を完全に明るく保ちます。
- **ESP tools** — ChestEsp、MobEsp、PlayerEsp、PortalEspで壁越しに主要なエンティティを見ます。
- **XRay** — 関心のあるブロックのみをフィルタリングします（設定可能なブロックリスト）。
- **FreeCamera / Camera distance/no-clip** — カメラを分離するか、距離を調整して建築を偵察します。
- **Radar** — 近くのエンティティをミニマップ形式でオーバーレイ表示し、範囲を設定できます。

## ヒント

- 必要なモジュールのみを組み合わせる：すべてを一度に実行すると、動きが不自然に感じられることがあります。
- 新しいモジュールをテストするときは、サーバー側の保護をトリガーしないように、シングルプレイヤーワールドから始めてください。
- クライアントを最新の状態に保つ：新しいリリースには、Fabricのアップデートに対する安定性修正が含まれていることがよくあります。
