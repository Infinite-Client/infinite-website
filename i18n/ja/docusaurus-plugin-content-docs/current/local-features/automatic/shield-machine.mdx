# ShieldMachine
シールドマシンを自動的に構築します。

## 設定

### Width
シールドマシンの幅を設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `3`
* **最小値**: `1`
* **最大値**: `6`

---

### Height
シールドマシンの高さを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `3`
* **最小値**: `2`
* **最大値**: `5`

---

### Length
シールドマシンの長さを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `100`
* **最小値**: `0`
* **最大値**: `256`

---

### ForwardOffset
シールドマシンの前方オフセットを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `2`
* **最小値**: `1`
* **最大値**: `5`

---

### AutoPlaceFloor
シールドマシンの床ブロックを自動配置します。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `true`

---

### FloorBlock
シールドマシンの床ブロックを設定します。

#### 設定情報

* **タイプ**: `BlockList`
* **デフォルト値**: `[minecraft:cobblestone]`
* **項目数**: 1
* **リストのタイプ**: Block IDs
