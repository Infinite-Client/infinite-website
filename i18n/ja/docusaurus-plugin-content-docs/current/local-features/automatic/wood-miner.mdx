# WoodMiner
自動的に木材を採掘します。

## 設定

### SearchRadius
木材採掘の検索半径を設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `16`
* **最小値**: `8`
* **最大値**: `64`

---

### SearchHeight
木材採掘の検索高さを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `8`
* **最小値**: `4`
* **最大値**: `32`

---

### WoodTypes
木材採掘で採掘する木の種類のリスト。

#### 設定情報

* **タイプ**: `BlockList`
* **デフォルト値**: `[]`
* **項目数**: 0
* **リストのタイプ**: Block IDs
