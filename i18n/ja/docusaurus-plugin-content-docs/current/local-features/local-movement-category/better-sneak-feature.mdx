# BetterSneakFeature

スニーク時の挙動を改善し、崖からの落下防止などを強化します。

## 設定項目

### safeWalk

セーフウォーク

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### edgeDistance

エッジ距離

| タイプ   | デフォルト値 | 詳細情報                      |
| :------- | :----------- | :---------------------------- |
| `Double` | `0.1`        | 最小値: `0.01`, 最大値: `0.5` |

---
