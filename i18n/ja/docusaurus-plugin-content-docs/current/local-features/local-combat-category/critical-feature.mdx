# CriticalFeature

攻撃時に自動的にクリティカル判定を発生させます。モードによって検知リスクと挙動が異なります。

## 設定項目

### mode

判定モード

| タイプ          | デフォルト値 | 詳細情報                                 |
| :-------------- | :----------- | :--------------------------------------- |
| `EnumSelection` | `Packet`     | 選択肢: `Packet`, `MiniJump`, `FullJump` |
