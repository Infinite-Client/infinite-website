# ArcheryFeature

弓やクロスボウの弾道を予測し、エイムを補助します。重力や空気抵抗を考慮した正確な射撃をサポートします。

## 設定項目

### simulationMaxSteps

シミュレーション最大ステップ

| タイプ | デフォルト値 | 詳細情報                    |
| :----- | :----------- | :-------------------------- |
| `Int`  | `100`        | 最小値: `20`, 最大値: `200` |

---

### simulationPrecision

シミュレーション精度

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `20`         | 最小値: `10`, 最大値: `64` |

---

### maxReach

最大射程

| タイプ | デフォルト値 | 詳細情報                    |
| :----- | :----------- | :-------------------------- |
| `Int`  | `128`        | 最小値: `16`, 最大値: `256` |

---

### ignoreTerrain

地形を無視

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |

---

### onlyWhenLockOn

ロックオン時のみ

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |

---
