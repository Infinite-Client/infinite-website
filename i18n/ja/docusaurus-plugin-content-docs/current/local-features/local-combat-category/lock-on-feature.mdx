Lock On Feature
=======
ターゲットを自動的に追尾する機能です。PvPやMobとの戦闘において、エイムを正確に保つことができます。方式や追尾速度を細かく調整可能です。

## 設定項目

### range

範囲

| タイプ   | デフォルト値 | 詳細情報                       |
| :------- | :----------- | :----------------------------- |
| `Double` | `32.0`       | 最小値: `3.0`, 最大値: `256.0` |

---

### players

プレイヤーを対象

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### mobs

| タイプ    | デフォルト値 | 詳細情報 |

| `Boolean` | `true`       | -        |
