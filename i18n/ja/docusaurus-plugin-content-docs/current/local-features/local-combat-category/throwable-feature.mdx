Throwable Feature
=======
雪玉やエンダーパールなどの投擲アイテムの軌道を表示し、命中を補助します。

## 設定項目

### simulationMaxSteps

シミュレーション最大ステップ

| タイプ | デフォルト値 | 詳細情報                    |
| :----- | :----------- | :-------------------------- |
| `Int`  | `100`        | 最小値: `20`, 最大値: `200` |

---

### simulationPrecision

シミュレーション精度

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `20`         | 最小値: `10`, 最大値: `64` |

---

### maxReach

最大射程

| タイプ | デフォルト値 | 詳細情報                    |
| :----- | :----------- | :-------------------------- |
| `Int`  | `64`         | 最小値: `16`, 最大値: `128` |

---

### onlyWhenLockOn

ロックオン時のみ

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |
