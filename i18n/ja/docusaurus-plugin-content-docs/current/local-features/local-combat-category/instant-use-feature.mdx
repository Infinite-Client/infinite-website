# InstantUseFeature

回復アイテムやブロックなどを、瞬時に使用・設置できるようにします。

## 設定項目

### mode

モード

| タイプ          | デフォルト値 | 詳細情報                                   |
| :-------------- | :----------- | :----------------------------------------- |
| `EnumSelection` | `RightClick` | 選択肢: `RightClick`, `FastPlace`, `Timed` |

---

### customDelay

カスタム遅延

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `2`          | 最小値: `0`, 最大値: `20` |

---

### fastPlaceSpeed

高速設置速度

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `3`          | 最小値: `1`, 最大値: `10` |

---

### allowCrystals

クリスタルを許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### allowExpBottles

経験値ボトルを許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### allowBlocks

ブロックを許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### allowThrowables

投擲アイテムを許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### allowFood

食べ物を許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |

---

### allowAll

すべてのアイテムを許可

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |

---

### customItems

カスタムアイテム

| タイプ     | デフォルト値 | 詳細情報 |
| :--------- | :----------- | :------- |
| `ItemList` | `[]`         | 選択肢:  |

---

### onlyWhenLockOn

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `false`      | -        |

---

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
