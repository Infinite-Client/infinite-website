# AntiCheat
アンチチートが特定の行動を検出するのを防ぎます。

## 設定

### EnableForQuickMove
クイックムーブメントのアンチチートを有効にします。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `false`

---

### QuickMoveTolerance
クイックムーブメントのアンチチートの許容範囲を設定します。

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `0.05`
* **最小値**: `0.0`
* **最大値**: `0.3`

---

### QuickMoveLatencyMultiplier
クイックムーブメントのアンチチートの遅延乗数を設定します。

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `0.5`
* **最小値**: `0.0`
* **最大値**: `1.0`
