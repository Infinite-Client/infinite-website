# AutoLeave

特定の条件でサーバーを自動的に離脱します。

## 設定

### HpThreshold

サーバーを離脱するHPパーセント。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `6`
- **現在の値**: `6`
- **最小値**: `0`
- **最大値**: `20`

---

### TotemThreshold

自動離脱をトリガーする残りトーテム数。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `0`
- **現在の値**: `0`
- **最小値**: `0`
- **最大値**: `11`

---

### Method

サーバーを離脱する方法。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `Normal`
- **現在の値**: `Normal`
- **選択肢**: `Normal`, `InvalidChat`, `SelfHurt`
