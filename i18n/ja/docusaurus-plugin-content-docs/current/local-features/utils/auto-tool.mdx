# AutoTool

採掘に最適なツールに自動切り替えします。

## 設定

### Method

最適なツールを選択する方法。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `HotBar`
- **現在の値**: `HotBar`
- **選択肢**: `Swap`, `HotBar`

---

### FineToolStrategy

最適なツールを選択する際の詳細な戦略。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `SharpTool`
- **現在の値**: `SharpTool`
- **選択肢**: `Shears`, `SharpTool`, `Hand`

---

### SwitchDelay

ツールを切り替えるまでのティック遅延を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `5`
- **現在の値**: `5`
- **最小値**: `0`
- **最大値**: `20`
