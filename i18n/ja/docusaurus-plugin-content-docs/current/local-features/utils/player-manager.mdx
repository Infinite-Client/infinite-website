# PlayerManager
フレンドとエネミーのリストを管理します。

## 設定

### Friends
フレンドとして扱うプレイヤーのリスト。

#### 設定情報

* **タイプ**: `PlayerList`
* **デフォルト値**: `[]`
* **項目数**: 0
* **リストのタイプ**: Player Names

---

### Enemies
エネミーとして扱うプレイヤーのリスト。

#### 設定情報

* **タイプ**: `PlayerList`
* **デフォルト値**: `[]`
* **項目数**: 0
* **リストのタイプ**: Player Names
