# BackPackManager
バックパック内のアイテムを管理します。

## 設定

### SortEnabled
バックパック内のアイテムを自動ソートします。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `true`

---

### AutoMoveToBackpack
収集したアイテムをバックパックに自動移動します。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `true`

---

### AutoReplenishHotbar
ホットバーのアイテムをバックパックから自動補充します。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `true`

---

### SortInterval
自動ソートの間隔（ティック）。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `100`
* **最小値**: `20`
* **最大値**: `1200`
