# NoAttack

攻撃しないエンティティのリスト。

## 設定

### ProtectedEntities

攻撃しないエンティティのリスト。

#### 設定情報

- **タイプ**: `EntityList`
- **デフォルト値**: `[minecraft:villager, minecraft:wolf, minecraft:cat]`
- **現在の値**: `[minecraft:villager, minecraft:wolf, minecraft:cat]`
- **項目数**: 3
- **リストのタイプ**: Entity IDs
