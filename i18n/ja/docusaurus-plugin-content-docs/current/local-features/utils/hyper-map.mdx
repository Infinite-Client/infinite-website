# HyperMap

画面にミニマップを表示します。

## 設定

### Radius

ミニマップ表示の半径を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `32`
- **最小値**: `5`
- **最大値**: `256`

---

### Height

ミニマップに表示する垂直高さ範囲を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `8`
- **最小値**: `1`
- **最大値**: `32`

---

### Margin

ミニマップUIの画面端からのマージンを設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `4`
- **最小値**: `0`
- **最大値**: `40`

---

### Size

ミニマップUIのサイズを設定します（画面短辺の%）。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `40`
- **最小値**: `5`
- **最大値**: `100`

---

### Mode

ミニマップの表示モードを設定します（FlatまたはSolid）。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `Flat`
- **選択肢**: `Flat`, `Solid`

---

### RenderTerrain

ミニマップに地形をレンダリングします。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---

### UseShading

ミニマップの地形レンダリングにシェーディング効果を適用します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---
