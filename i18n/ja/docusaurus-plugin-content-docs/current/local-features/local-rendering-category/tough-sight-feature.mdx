# ToughSightFeature

盲目や吐き気などのステータス効果による視覚的な妨害を無効化します。

## 設定項目

### antiNausea

吐き気耐性

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### antiBlindness

盲目耐性

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### antiDarkness

暗闇耐性

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---
