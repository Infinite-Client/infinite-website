# StableSightFeature

カメラの揺れを抑え、視点を安定させます。

## 設定項目

### cameraDistance

カメラ距離

| タイプ  | デフォルト値 | 詳細情報                        |
| :------ | :----------- | :------------------------------ |
| `Float` | `10.0`       | 最小値: `-10.0`, 最大値: `64.0` |

---

### ignoreTerrain

地形を無視

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |
