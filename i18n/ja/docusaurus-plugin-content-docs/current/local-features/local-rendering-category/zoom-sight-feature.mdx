# ZoomSightFeature

視界をズーム（拡大）します。望遠鏡のような効果を、キー操作ひとつでスムーズに利用可能です。

## 設定項目

### smoothZoom

スムーズズーム

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### zoomLevel

ズームレベル

| タイプ  | デフォルト値 | 詳細情報                      |
| :------ | :----------- | :---------------------------- |
| `Float` | `4.0`        | 最小値: `1.0`, 最大値: `50.0` |

---

### zoomStep

ズームステップ

| タイプ  | デフォルト値 | 詳細情報                      |
| :------ | :----------- | :---------------------------- |
| `Float` | `1.1`        | 最小値: `1.01`, 最大値: `2.0` |

---

### sensitivityReduction

感度低下

| タイプ  | デフォルト値 | 詳細情報                     |
| :------ | :----------- | :--------------------------- |
| `Float` | `1.0`        | 最小値: `0.0`, 最大値: `1.0` |

---

### autoReleaseOnMove

移動時に自動解除

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### maxZoom

最大ズーム

| タイプ  | デフォルト値 | 詳細情報                        |
| :------ | :----------- | :------------------------------ |
| `Float` | `50.0`       | 最小値: `10.0`, 最大値: `256.0` |

---
