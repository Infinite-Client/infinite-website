# ClearSightFeature

フォグ（霧）やオーバーレイ（水中、炎など）を除去し、視界をクリアにします。

## 設定項目

### antiFog

フォグ除去

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### antiOverlay

オーバーレイ除去

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### antiFovChange

FOV変化耐性

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### fovMaxIncrease

最大FOV増加

| タイプ  | デフォルト値 | 詳細情報                     |
| :------ | :----------- | :--------------------------- |
| `Float` | `0.12`       | 最小値: `0.0`, 最大値: `0.5` |

---

### fovIntensity

FOV強度

| タイプ  | デフォルト値 | 詳細情報                      |
| :------ | :----------- | :---------------------------- |
| `Float` | `8.0`        | 最小値: `1.0`, 最大値: `20.0` |

---
