# UltraUiFeature

スタイリッシュで高機能なカスタムHUDを表示します。

## 設定項目

### hotbarUi

ホットバーUI

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### topBoxUi

上ボックスUI

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### leftBoxUi

左ボックスUI

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### rightBoxUi

右ボックスUI

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### crosshairUi

十字線UI

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### barHeight

バーの高さ

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `24`         | 最小値: `8`, 最大値: `32` |

---

### padding

UIの余白

| タイプ | デフォルト値 | 詳細情報                 |
| :----- | :----------- | :----------------------- |
| `Int`  | `4`          | 最小値: `0`, 最大値: `8` |

---

### alpha

UIの透明度

| タイプ  | デフォルト値 | 詳細情報                     |
| :------ | :----------- | :--------------------------- |
| `Float` | `0.8`        | 最小値: `0.0`, 最大値: `1.0` |

---
