# SuperFly
飛行能力を強化します。

## 設定

### Method
SuperFlyの方法

#### 設定情報

* **タイプ**: `Enum`
* **デフォルト値**: `Acceleration`
* **現在の値**: `Acceleration`
* **選択肢**: `Acceleration`, `Rocket`, `CreativeFlight`

---

### Power
飛行パワー。

#### 設定情報

* **タイプ**: `Float`
* **デフォルト値**: `1.0`
* **現在の値**: `1.0`
* **最小値**: `0.5`
* **最大値**: `5.0`

---

### KeepFlying
強制キャンセルされても飛行を維持します。

#### 設定情報

* **タイプ**: `Boolean`
* **デフォルト値**: `true`
* **現在の値**: `true`
