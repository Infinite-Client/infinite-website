# Freeze

サーバーへの移動パケット送信を停止し、テレポートを可能にします。

## 設定

### Duration

Freezeが自動無効化される最大時間（秒）。0で無制限。

#### 設定情報

- **タイプ**: `Float`
- **デフォルト値**: `0.0`
- **現在の値**: `0.0`
- **最小値**: `0.0`
- **最大値**: `600.0`

---

### PacketLimit

Freezeが自動再起動される前に蓄積するパケットの最大数。0で無制限。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `0`
- **現在の値**: `0`
- **最小値**: `0`
- **最大値**: `500`
