# SuperSprint

空腹時や後ろ向き移動時でもスプリントを強制します。

## 設定

### OnlyWhenForward

前進時のみスプリントを有効にします。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`
- **現在の値**: `true`

---

### EvenIfHungry

空腹が低い時でもスプリントを強制します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `false`
- **現在の値**: `false`
