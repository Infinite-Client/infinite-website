# FeatherWalk

特定のブロック上でジャンプとスプリントを防止します。

## 設定

### AllowedBlocks

ジャンプとスプリントを抑制するブロックIDのリスト。

#### 設定情報

- **タイプ**: `BlockList`
- **デフォルト値**: `[minecraft:farmland]`
- **現在の値**: `[minecraft:farmland]`
- **項目数**: 1
- **リストのタイプ**: Block IDs

---

### DisableJump

設定したブロック上でジャンプを無効化します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`
- **現在の値**: `true`

---

### DisableSprint

設定したブロック上でスプリントを無効化します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`
- **現在の値**: `true`
