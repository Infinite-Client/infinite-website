# LinearBreak

クロスヘアでターゲットしたすべてのブロックを記憶し、パケット送信で迅速に破壊します。

## 設定

### Range

リニアブレイクの範囲を設定します。

#### 設定情報

- **タイプ**: `Double`
- **デフォルト値**: `5.0`
- **最小値**: `1.0`
- **最大値**: `6.0`

---

### MaxBlocks

リニアブレイクで一度に破壊する最大ブロック数を設定します。

#### 設定情報

- **タイプ**: `Double`
- **デフォルト値**: `64.0`
- **最小値**: `1.0`
- **最大値**: `500.0`

---

### SwingHand

リニアブレイク中に手を振るかどうかを設定します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---

### IgnoreHotbarChange

リニアブレイク中にホットバーの変更を無視するかどうかを設定します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `false`
