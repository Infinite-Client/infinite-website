# HoverVehicle

乗り物をホバー車両のように動作させ、水上を高速移動可能にします。

## 設定

### MaxSpeed

ホバー車両の速度乗数を設定します。

#### 設定情報

- **タイプ**: `Float`
- **デフォルト値**: `20.0`
- **最小値**: `1.0`
- **最大値**: `200.0`

---

### Acceleration

ホバー車両の加速パワーを設定します。

#### 設定情報

- **タイプ**: `Float`
- **デフォルト値**: `0.5`
- **最小値**: `0.05`
- **最大値**: `2.0`
