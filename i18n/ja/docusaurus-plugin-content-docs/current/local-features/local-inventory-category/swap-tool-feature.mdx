# SwapToolFeature

破壊するブロックに合わせて、最適なツールに自動的に持ち替えます。

## 設定項目

### method

方法

| タイプ          | デフォルト値 | 詳細情報                 |
| :-------------- | :----------- | :----------------------- |
| `EnumSelection` | `HotBar`     | 選択肢: `Swap`, `HotBar` |

---

### fineToolStrategy

精密ツール戦略

| タイプ          | デフォルト値 | 詳細情報                              |
| :-------------- | :----------- | :------------------------------------ |
| `EnumSelection` | `SharpTool`  | 選択肢: `Shears`, `SharpTool`, `Hand` |

---

### switchDelay

切り替え遅延

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `5`          | 最小値: `0`, 最大値: `20` |
