# AutoArmorFeature

インベントリ内の最適な防具を自動的に装備します。耐久値が低下した際のエリトラ切り替え機能なども搭載しています。

## 設定項目

### autoEquip

自動装備を有効にする

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### elytraSwitch

エリトラ自動切り替え

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### durabilityThreshold

耐久値のしきい値

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `5`          | 最小値: `0`, 最大値: `100` |

---

### delay

装備遅延

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `2`          | 最小値: `1`, 最大値: `20` |

---
