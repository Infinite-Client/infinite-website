# ItemRestockFeature

アイテムの自動補充を行います。

## 設定項目

### delayProperty

補充遅延

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `3`          | 最小値: `1`, 最大値: `20` |

---

### thresholdProperty

補充のしきい値

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `8`          | 最小値: `1`, 最大値: `64` |

---
