# ContainerUtilFeature

コンテナ操作を便利にする各種機能（ホットバー回転など）を提供します。

## 設定項目

### autoRocket

ロケット自動補充

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### instantStop

インスタント停止

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### hotbarRotate

ホットバー回転

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---
