# MaceAssist

メイス武器の使用をアシストします。

## 設定

### SearchTick

ターゲットを検索するティック間隔を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `128`
- **最小値**: `0`
- **最大値**: `256`

---

### ReactTick

ターゲットに反応するティック間隔を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `10`
- **最小値**: `0`
- **最大値**: `10`

---

### Method

メイスアシストの方法を設定します。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `Linear`
- **選択肢**: `Linear`, `EaseIn`, `EaseOut`, `EaseInOut`, `Immediate`
