# CounterAttack

ダメージを受けたときに自動的に攻撃します。

## 設定

### ReactionTick

攻撃に反応するティック間隔を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `4`
- **最小値**: `0`
- **最大値**: `10`

---

### ProcessTick

カウンター攻撃を処理するティック間隔を設定します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `4`
- **最小値**: `0`
- **最大値**: `10`

---

### Randomizer

カウンター攻撃のタイミングにランダム性を追加します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `2`
- **最小値**: `0`
- **最大値**: `10`

---

### Method

カウンター攻撃の方法を設定します。

#### 設定情報

- **タイプ**: `Enum`
- **デフォルト値**: `Linear`
- **選択肢**: `Linear`, `EaseIn`, `EaseOut`, `EaseInOut`, `Immediate`

---

### AimSpeed

カウンター攻撃のエイム速度を設定します。

#### 設定情報

- **タイプ**: `Double`
- **デフォルト値**: `5.0`
- **最小値**: `1.0`
- **最大値**: `10.0`
