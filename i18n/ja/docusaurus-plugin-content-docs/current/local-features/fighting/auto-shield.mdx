# AutoShield

攻撃に対して自動的にシールドします。

## 設定

### DetectionRange

incoming attacksを検出する範囲を設定します。

#### 設定情報

- **タイプ**: `Double`
- **デフォルト値**: `4.0`
- **最小値**: `1.0`
- **最大値**: `10.0`

---

### AutoEquip

攻撃検出時にシールドを自動装備します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---

### PredictionTicks

敵の攻撃を予測するティック数。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `10`
- **最小値**: `1`
- **最大値**: `40`

---

### MeleeRangeMultiplier

近接攻撃検出範囲の乗数。

#### 設定情報

- **タイプ**: `Double`
- **デフォルト値**: `1.5`
- **最小値**: `0.5`
- **最大値**: `3.0`
