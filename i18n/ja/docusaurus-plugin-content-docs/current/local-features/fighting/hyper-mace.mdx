# HyperMace
メイスのダメージを大幅に増加させます。

## 設定

### DamageBoost
Hyper Maceによるダメージブースト。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `4`
* **最小値**: `0`
* **最大値**: `10`

---

### FallMultiply
Hyper Mace効果によるダメージ倍率

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `1.5`
* **最小値**: `1.0`
* **最大値**: `5.0`

---

### FallDistance
Hyper Mace効果が発動させる最小落下距離。

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `500.0`
* **最小値**: `1.0`
* **最大値**: `1000.0`
