# ArmorManager

最適なアーマーを自動装備し、エリトラを管理します。

## 設定

### AutoEquip

最適な利用可能なアーマーを自動装備します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---

### ElytraSwitch

空中にいるときにエリトラに自動切り替えします。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`

---

### DurabilityThreshold

このパーセント（%）以下の耐久度の鎧を無視します。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `5`
- **最小値**: `0`
- **最大値**: `100`
