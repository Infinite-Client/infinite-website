# ImpactAttack
攻撃キーを押したときに視線上のターゲットを自動的に攻撃します。

## 設定

### Range
自動攻撃の最大攻撃範囲を設定します。

#### 設定情報

* **タイプ**: `Float`
* **デフォルト値**: `4.2`
* **現在の値**: `4.2`
* **最小値**: `3.0`
* **最大値**: `7.0`

---

### CPS
自動攻撃のクリック数/秒を設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `10`
* **現在の値**: `10`
* **最小値**: `1`
* **最大値**: `20`
