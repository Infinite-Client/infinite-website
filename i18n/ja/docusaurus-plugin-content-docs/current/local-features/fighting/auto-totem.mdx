# AutoTotem

体力が低いときに自動的にトーテムを装備します。

## 設定

### Hp

トーテムを装備するHP。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `10`
- **最小値**: `1`
- **最大値**: `20`

---

### MinDelayMs

オートトーテムの最小遅延（ミリ秒）。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `60`
- **最小値**: `0`
- **最大値**: `500`

---

### MaxDelayMs

オートトーテムの最大遅延（ミリ秒）。

#### 設定情報

- **タイプ**: `Int`
- **デフォルト値**: `180`
- **最小値**: `0`
- **最大値**: `500`
