# HyperUi
強化されたUI

## 設定

### EasingSpeed
UIアニメーションのイージング速度を設定します。

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `0.25`
* **現在の値**: `0.25`
* **最小値**: `0.0`
* **最大値**: `1.0`

---

### Transparency
Hyper UIの透明度を設定します。

#### 設定情報

* **タイプ**: `Double`
* **デフォルト値**: `0.5`
* **現在の値**: `0.5`
* **最小値**: `0.0`
* **最大値**: `1.0`

---

### Height
Hyper UIの高さを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `24`
* **現在の値**: `24`
* **最小値**: `8`
* **最大値**: `32`

---

### Padding
Hyper UIのパディングを設定します。

#### 設定情報

* **タイプ**: `Int`
* **デフォルト値**: `4`
* **現在の値**: `4`
* **最小値**: `0`
* **最大値**: `16`

---
