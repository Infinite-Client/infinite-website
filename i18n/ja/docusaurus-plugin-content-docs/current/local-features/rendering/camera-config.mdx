# CameraConfig

カメラ距離、クリッピング、アンチハートティルトを設定します。

## 設定

### CameraDistance

カメラとプレイヤーの距離を設定します。

#### 設定情報

- **タイプ**: `Float`
- **デフォルト値**: `8.0`
- **現在の値**: `8.0`
- **最小値**: `-1.0`
- **最大値**: `15.0`

---

### ClipBlock

カメラが地形を貫通するかどうかを設定します。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`
- **現在の値**: `true`

---

### AntiHurtTilt

ダメージ時にカメラが揺れなくなります。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `true`
- **現在の値**: `true`

---

### ExtraCamera

カメラがプレイヤーの向きに依存しなくなります。

#### 設定情報

- **タイプ**: `Boolean`
- **デフォルト値**: `false`
- **現在の値**: `false`
