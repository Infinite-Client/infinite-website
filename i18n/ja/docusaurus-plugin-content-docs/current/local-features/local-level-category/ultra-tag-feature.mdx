Ultra Tag Feature
=======
エンティティの上に詳細な情報を表示するタグを追加します。

## 設定項目

### range

表示範囲

| タイプ | デフォルト値 | 詳細情報                    |
| :----- | :----------- | :-------------------------- |
| `Int`  | `64`         | 最小値: `16`, 最大値: `256` |

---

### showMobs

モブを表示する

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### showPlayers

プレイヤーを表示する

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### showItems

アイテムを表示する

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### showEquipments

装備を表示する

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### fadeStart

フェードアウトが始まる距離

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `30`         | 最小値: `1`, 最大値: `256` |

---

### fadeEnd

フェードアウトが終了する距離

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `60`         | 最小値: `1`, 最大値: `256` |

---

### itemHideAngle

アイテムを隠す角度

| タイプ | デフォルト値 | 詳細情報                  |
| :----- | :----------- | :------------------------ |
| `Int`  | `20`         | 最小値: `0`, 最大値: `90` |
