# FastBreakFeature

ブロックを通常よりも速く破壊できるようにします（サバイバルモード用）。

## 設定項目

### safeMode

セーフモード

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### interval

間隔

| タイプ | デフォルト値 | 詳細情報                 |
| :----- | :----------- | :----------------------- |
| `Int`  | `2`          | 最小値: `0`, 最大値: `5` |

---

### thresholdPercentage

しきい値パーセンテージ

| タイプ   | デフォルト値 | 詳細情報                       |
| :------- | :----------- | :----------------------------- |
| `Double` | `20.0`       | 最小値: `0.0`, 最大値: `100.0` |

---

### thresholdTick

しきい値ティック

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `5`          | 最小値: `1`, 最大値: `100` |

---
