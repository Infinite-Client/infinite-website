ESP Feature
=======
壁越しにエンティティ（プレイヤー、Mob、アイテム）を視認できるようにします。不透明度や距離に応じたフェードアウトも設定可能です。

## 設定項目

### playerEsp

プレイヤー ESP

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### mobEsp

Mob ESP

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |

---

### itemEsp

アイテム ESP

| タイプ    | デフォルト値 | 詳細情報 |
| :-------- | :----------- | :------- |
| `Boolean` | `true`       | -        |
