# XRayFeature

特定のブロックのみを透過表示し、鉱石探しなどを効率化します。透過率やターゲットとなるブロックを自由に選択できます。

## 設定項目

### method

方式

| タイプ          | デフォルト値 | 詳細情報                                                                 |
| :-------------- | :----------- | :----------------------------------------------------------------------- |
| `EnumSelection` | `Full`       | 選択肢: `OnlyExposed`, `Full`, `TransparencyExposed`, `TransparencyFull` |

---

### transparency

透明度

| タイプ  | デフォルト値 | 詳細情報                     |
| :------ | :----------- | :--------------------------- |
| `Float` | `0.5`        | 最小値: `0.0`, 最大値: `1.0` |
