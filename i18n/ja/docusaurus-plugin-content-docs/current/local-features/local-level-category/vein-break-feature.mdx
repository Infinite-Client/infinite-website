# VeinBreakFeature

つながっている同じ種類の鉱石を一括で破壊します（連鎖破壊）。

## 設定項目

| :---------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `BlockList` | `[minecraft:ancient_debris, minecraft:coal_ore, minecraft:copper_ore, minecraft:deepslate_coal_ore, minecraft:deepslate_copper_ore, minecraft:deepslate_diamond_ore, minecraft:deepslate_emerald_ore, minecraft:deepslate_gold_ore, minecraft:deepslate_iron_ore, minecraft:deepslate_lapis_ore, minecraft:deepslate_redstone_ore, minecraft:diamond_ore, minecraft:emerald_ore, minecraft:gold_ore, minecraft:iron_ore, minecraft:lapis_ore, minecraft:nether_gold_ore, minecraft:nether_quartz_ore, minecraft:redstone_ore]` | 選択肢: `minecraft:ancient_debris`, `minecraft:coal_ore`, `minecraft:copper_ore`, `minecraft:deepslate_coal_ore`, `minecraft:deepslate_copper_ore`, `minecraft:deepslate_diamond_ore`, `minecraft:deepslate_emerald_ore`, `minecraft:deepslate_gold_ore`, `minecraft:deepslate_iron_ore`, `minecraft:deepslate_lapis_ore`, `minecraft:deepslate_redstone_ore`, `minecraft:diamond_ore`, `minecraft:emerald_ore`, `minecraft:gold_ore`, `minecraft:iron_ore`, `minecraft:lapis_ore`, `minecraft:nether_gold_ore`, `minecraft:nether_quartz_ore`, `minecraft:redstone_ore` |

### breakRange

破壊範囲

| タイプ   | デフォルト値 | 詳細情報                     |
| :------- | :----------- | :--------------------------- |
| `Double` | `5.0`        | 最小値: `1.0`, 最大値: `6.0` |

---

### maxBlocks

最大ブロック数

| タイプ | デフォルト値 | 詳細情報                   |
| :----- | :----------- | :------------------------- |
| `Int`  | `64`         | 最小値: `1`, 最大値: `500` |
