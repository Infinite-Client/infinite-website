# FAQ

## Infinite Clientはすべてのサーバーで許可されていますか？

いいえ。多くのサーバーでは、クライアントサイドの自動化や視覚補助が禁止されています。モジュールを有効にする前に、各サーバーのルールを読み、それに従ってください。

## 他のMinecraftバージョンで動作しますか？

現在のビルドは `1.21.10` を対象としています。他のバージョンで使用することはサポートされておらず、クラッシュする可能性があります。対応するリリースを待つか、正しいバージョンに対してソースからビルドしてください。

## 設定はどこにありますか？

クライアントを開くためのコントロールとモジュールごとの設定は、Minecraftの「設定 → コントロール」内、およびゲーム内のInfinite Clientメニューにあります。すべてをリセットするには、`mods` フォルダからjarファイルを削除してください。

## 自分でビルドできますか？

はい。リポジトリをクローンし、Java 21+をインストールし、`./gradlew build` を実行してください。jarファイルは `build/libs/` に生成されます。

## 問題が発生しましたか？

- Fabric LoaderとFabric APIを `1.21.10` 用に再インストールしてください。
- 競合を除外するために、他のModなしで新しいシングルプレイヤーワールドでテストしてください。
- 既知の問題やパッチについては、GitHubの最新リリースノートを確認してください。
