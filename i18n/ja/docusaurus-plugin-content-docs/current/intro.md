# Infinite Client

クリーンでミニマルなインターフェースを備えた、Minecraft 1.21.10用の軽量Fabricユーティリティクライアント。

## Infinite Clientとは？

- 建築、移動、戦闘、視認性アシストに特化した機能セット。
- Fabricベース（ローダー0.17.x、Fabric API 0.135.0+1.21.10）で、バニラのパフォーマンスに近い状態を維持。
- シンプルなUIと、煩雑さを避けた合理的なデフォルト設定。

> クライアントは責任を持って使用し、参加するサーバーのルールに従ってください。

## クイックスタート

1.  Minecraft `1.21.10` 用の [Fabricローダー](https://fabricmc.net/use/installer/) をインストールします。
2.  対応する [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api) を `mods` フォルダに追加します。
3.  最新のInfinite Client jarを [ダウンロードページ](/downloads) から `.minecraft/mods` にドロップします。
4.  Fabricプロファイルでゲームを起動し、Infinite Clientメニューを開いてモジュールを切り替えます。
