# インストール

Infinite ClientはMinecraft Java Edition `26.1.2` とFabric上で動作します。

## 必要条件

- Minecraft `26.1.2` (Java Edition)
- Fabric Loader `0.19.2` 以上
- [Fabric API 0.146.0+26.1.2](https://www.curseforge.com/minecraft/mc-mods/fabric-api)

## インストール手順

1.  [Fabricインストーラー](https://fabricmc.net/use/installer/) をダウンロードして実行し、`26.1.2` プロファイルを選択します。
2.  `fabric-api-0.146.0+26.1.2.jar` を `.minecraft/mods` にドロップします。
3.  最新のInfinite Client jarを [ダウンロードページ](/downloads) から `.minecraft/mods` にドロップします。
4.  Fabricプロファイルでゲームを起動します。タイトル画面のModリストにInfinite Clientが表示されるはずです。

## 更新

- 既存の `infinite-*.jar` を `.minecraft/mods` 内の最新リリースに置き換えます。
- クラッシュやミキシン不足を避けるため、Fabric LoaderとFabric APIは対応する `26.1.2` バージョンを維持してください。

## アンインストール

- Infinite Clientのjarファイルを `.minecraft/mods` から削除します。
- Infinite Clientのために追加した補助Modも同様に削除してください。
