# クライアントの使用方法

Infinite Clientはシンプルさを追求しています。必要なものだけを有効にし、画面をきれいに保ちます。

## メニューを開く

- Minecraftを開き、ワールドまたはサーバーに参加します。
- クライアントメニューを開くには、「設定 → コントロール → キー割り当て → Infinite Client」に表示されているキー割り当てを使用します。
- 他のModと競合する場合は、ここでメニューキーを再割り当てしてください。

## モジュールの切り替え

- モジュールは目的別（移動、戦闘、建築、レンダリング）にグループ化されています。
- モジュールをクリックしてオン/オフを切り替えます。一部のモジュールでは、微調整用のスライダーやトグルが表示されます。
- 世界や動きにどのような影響を与えるかを学ぶために、一度に1つのモジュールから始めてください。

## プロファイルとプリセット

- サバイバル用には軽量なモジュールセットを、建築用には別のセットをアクティブに保ちます。
- サーバーを切り替える際は、自動戦闘やブロック配置機能がそのサーバーのルールに違反していないか再確認してください。

## トラブルシューティング

- モジュールを切り替えても何も起こらない: Fabric APIがインストールされたFabric `1.21.10` プロファイルを使用していることを確認してください。
- キー割り当ての競合: バニラ設定または他のModで重複しているキー割り当てをクリアしてください。
- レンダリングの不具合: 競合する視覚Mod（例：他のフルブライト/X線スタイルの調整）を無効にして再起動してください。
