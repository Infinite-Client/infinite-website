# XRay
Allows seeing through blocks to find ores and hidden structures.

## Settings

### Method
XRay operation mode

#### Setting Information

* **Type**: `Enum`
* **Default Value**: `Normal`
* **Options**: `Normal`, `OnlyExposed`

---
