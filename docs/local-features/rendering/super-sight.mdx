# SuperSight
Maximizes game brightness and disables blindness effects.

## Settings

### NightVision
Enable night vision

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `true`
* **Current Value**: `true`

---

### AntiBlind
Disables blindness and darkness effects.

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `true`
* **Current Value**: `true`
