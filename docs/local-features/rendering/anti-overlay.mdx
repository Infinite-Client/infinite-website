# AntiOverlay

Removes various screen overlays.

## Settings

### NoPumpkinOverlay

Removes pumpkin overlay.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### NoDarknessOverlay

Removes darkness overlay (e.g., blindness effect).

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### NoLiquidOverlay

Removes overlay when in water/lava.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### NoFogOverlay

Disables fog effect.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`
