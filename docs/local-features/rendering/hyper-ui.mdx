# HyperUi

Configures the Hyper UI display.

## Settings

### EasingSpeed

Sets the easing speed for UI animations.

#### Setting Information

- **Type**: `Double`
- **Default Value**: `0.25`
- **Minimum Value**: `0.0`
- **Maximum Value**: `1.0`

---

### Transparency

Sets the transparency level of the Hyper UI.

#### Setting Information

- **Type**: `Double`
- **Default Value**: `0.5`
- **Minimum Value**: `0.0`
- **Maximum Value**: `1.0`

---

### Height

Sets the height of the Hyper UI.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `24`
- **Minimum Value**: `8`
- **Maximum Value**: `32`

---

### Padding

Sets the padding of the Hyper UI.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `4`
- **Minimum Value**: `0`
- **Maximum Value**: `16`
