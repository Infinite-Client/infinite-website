# DetailInfo

Displays detailed information about the targeted block or entity.

## Settings

### BlockInfo

Displays block information.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`

---

### InnerChest

Also retrieves chest contents.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`

---

### EntityInfo

Displays entity information.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`

---

### PaddingTop

Top padding.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `0`
- **Minimum Value**: `0`
- **Maximum Value**: `100`

---

### Reach

Maximum distance to display information.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `20.0`
- **Minimum Value**: `10.0`
- **Maximum Value**: `100.0`

---

### Width

Sets the width of the widget.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `50`
- **Minimum Value**: `25`
- **Maximum Value**: `100`
