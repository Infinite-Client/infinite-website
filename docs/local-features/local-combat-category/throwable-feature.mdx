# ThrowableFeature

Displays the trajectory of throwable items like Snowballs or Ender Pearls to improve accuracy.

## Properties

### simulationMaxSteps

Simulation Steps

| Type  | Default | Details                       |
| :---- | :------ | :---------------------------- |
| `Int` | `100`   | Minimum: `20`, Maximum: `200` |

---

### simulationPrecision

Simulation Precision

| Type  | Default | Details                      |
| :---- | :------ | :--------------------------- |
| `Int` | `20`    | Minimum: `10`, Maximum: `64` |

---

### maxReach

Max Reach

| Type  | Default | Details                       |
| :---- | :------ | :---------------------------- |
| `Int` | `64`    | Minimum: `16`, Maximum: `128` |

---

### onlyWhenLockOn

Only When Lock-On

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `false` | -       |
