# Gunner

Manages crossbow firing and reloading.

## Settings

### FireMode

The firing mode for shooting.

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `FULL_AUTO`
- **Current Value**: `FULL_AUTO`
- **Options**: `SEMI_AUTO`, `FULL_AUTO`

---

### FastReload

If enabled, reloading become fast

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `false`
- **Current Value**: `false`

---

### ChangeMode

How to toggle reload and shot

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `Fixed`
- **Current Value**: `Fixed`
- **Options**: `Fixed`, `Toggle`

---

### AdditionalInterval

Additional interval for full-auto

#### Setting Information

- **Type**: `Int`
- **Default Value**: `3`
- **Current Value**: `3`
- **Minimum Value**: `0`
- **Maximum Value**: `10`
