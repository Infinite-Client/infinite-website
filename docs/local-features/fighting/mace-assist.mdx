# MaceAssist
Assists with using the mace weapon.

## Settings

### SearchTick
Sets the tick interval for searching for targets.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `128`
* **Current Value**: `128`
* **Minimum Value**: `0`
* **Maximum Value**: `256`

---

### ReactTick
Sets the tick interval for reacting to targets.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `10`
* **Current Value**: `10`
* **Minimum Value**: `0`
* **Maximum Value**: `10`

---

### Method
Sets the method for mace assist.

#### Setting Information

* **Type**: `Enum`
* **Default Value**: `Linear`
* **Current Value**: `Linear`
* **Options**: `Linear`, `EaseIn`, `EaseOut`, `EaseInOut`, `Immediate`
