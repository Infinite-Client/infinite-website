# ImpactAttack

Automatically attacks the target you are looking at when the attack key is pressed.

## Settings

### Range

Sets the maximum attack range for automatic attacks.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `4.2`
- **Current Value**: `4.2`
- **Minimum Value**: `3.0`
- **Maximum Value**: `7.0`

---

### CPS

Sets the clicks per second for automatic attacks.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `10`
- **Current Value**: `10`
- **Minimum Value**: `1`
- **Maximum Value**: `20`
