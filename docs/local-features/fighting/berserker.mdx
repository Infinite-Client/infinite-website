# Berserker

Increases attack speed and damage when health is low.

## Settings

### AttackRange

Sets the attack range for Berserker mode.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `7.5`
- **Current Value**: `7.5`
- **Minimum Value**: `5.0`
- **Maximum Value**: `10.0`
