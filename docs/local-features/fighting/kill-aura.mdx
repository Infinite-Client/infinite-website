# KillAura

Sets the maximum attack distance for entities.

## Settings

### Range

Sets the maximum attack distance for entities.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `4.2`
- **Current Value**: `4.2`
- **Minimum Value**: `3.0`
- **Maximum Value**: `7.0`

---

### Players

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

### Mobs
