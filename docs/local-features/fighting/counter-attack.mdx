# CounterAttack

Automatically attacks when damaged.

## Settings

### ReactionTick

Sets the tick interval for reacting to attacks.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `4`
- **Minimum Value**: `0`
- **Maximum Value**: `10`

---

### ProcessTick

Sets the tick interval for processing counter attacks.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `4`
- **Minimum Value**: `0`
- **Maximum Value**: `10`

---

### Randomizer

Adds randomness to counter attack timing.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `2`
- **Minimum Value**: `0`
- **Maximum Value**: `10`

---

### Method

Sets the method for counter attack.

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `Linear`
- **Options**: `Linear`, `EaseIn`, `EaseOut`, `EaseInOut`, `Immediate`

---

### AimSpeed

Sets the aim speed for counter attack.

#### Setting Information

- **Type**: `Double`
- **Default Value**: `5.0`
- **Minimum Value**: `1.0`
- **Maximum Value**: `10.0`
