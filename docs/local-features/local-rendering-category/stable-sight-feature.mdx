# StableSightFeature

Reduces camera shake and stabilizes your view.

## Properties

### cameraDistance

Camera Distance

| Type    | Default | Details                           |
| :------ | :------ | :-------------------------------- |
| `Float` | `10.0`  | Minimum: `-10.0`, Maximum: `64.0` |

---

### ignoreTerrain

Ignore Terrain

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |
