# ClearSightFeature

Removes fog and screen overlays (like fire or water) for a clear, unobstructed view.

## Properties

### antiFog

Anti-Fog

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### antiOverlay

Anti-Overlay

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### antiFovChange

Anti FOV Change

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### fovMaxIncrease

Max FOV Increase

| Type    | Default | Details                        |
| :------ | :------ | :----------------------------- |
| `Float` | `0.12`  | Minimum: `0.0`, Maximum: `0.5` |

---

### fovIntensity

FOV Intensity

| Type    | Default | Details                         |
| :------ | :------ | :------------------------------ |
| `Float` | `8.0`   | Minimum: `1.0`, Maximum: `20.0` |
