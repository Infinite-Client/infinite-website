| `Boolean` | `true`  | -       |

---

### crosshairUi

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

Bar Height

| Type  | Default | Details                     |
| :---- | :------ | :-------------------------- |

---
