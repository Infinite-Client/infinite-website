# PlayerManager

Manages lists of friends and enemies.

## Settings

### Friends

List of players to treat as friends.

#### Setting Information

- **Type**: `PlayerList`
- **Default Value**: `[]`
- **Item Count**: 0
- **List Type**: Player Names

---

### Enemies

List of players to treat as enemies.

#### Setting Information

- **Type**: `PlayerList`
- **Default Value**: `[]`
- **Item Count**: 0
- **List Type**: Player Names
