# BackPackManager

Manages items in the backpack.

## Settings

### SortEnabled

Automatically sorts items in the backpack.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### AutoMoveToBackpack

Automatically moves collected items to the backpack.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### AutoReplenishHotbar

Automatically replenishes items in the hotbar from the backpack.

#### Setting Information

- **Type**: `Boolean`
- **Default Value**: `true`
- **Current Value**: `true`

---

### SortInterval

Interval (in ticks) for automatic sorting.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `100`
- **Current Value**: `100`
- **Minimum Value**: `20`
- **Maximum Value**: `1200`
