# AutoTool

Automatically switches to the best tool for mining.

## Settings

### Method

Method for selecting the best tool.

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `HotBar`
- **Current Value**: `HotBar`
- **Options**: `Swap`, `HotBar`

---

### FineToolStrategy

Detailed strategy for selecting the best tool.

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `SharpTool`
- **Current Value**: `SharpTool`
- **Options**: `Shears`, `SharpTool`, `Hand`

---

### SwitchDelay

Sets the delay in ticks before switching tools.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `5`
- **Current Value**: `5`
- **Minimum Value**: `0`
- **Maximum Value**: `20`
