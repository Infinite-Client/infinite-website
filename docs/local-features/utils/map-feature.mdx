# MapFeature
Displays a mini-map on the screen.

## Settings

### ZoomLevel
Sets the zoom level of the mini-map.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `1`
* **Current Value**: `1`
* **Minimum Value**: `1`
* **Maximum Value**: `10`

---

### ShowPlayerCoordinates
Shows player coordinates on the mini-map.

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `true`
* **Current Value**: `true`
