# CaveHighlightFeature

Highlights caves and air pockets to make subterranean exploration easier.

## Properties

| Type                | Default                                                                                                                                                                    | Details                            |
| :------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------- |
| `BlockAndColorList` | `[BlockAndColor(blockId=minecraft:lava, color=553600256), BlockAndColor(blockId=minecraft:water, color=536888831), BlockAndColor(blockId=minecraft:air, color=553648127)]` | Default Count: 3 (Block-Color Map) |
