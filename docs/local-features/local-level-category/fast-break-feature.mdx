# FastBreakFeature

Enables faster block breaking in survival mode.

## Properties

### safeMode

Safe Mode

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### interval

Interval

| Type  | Default | Details                    |
| :---- | :------ | :------------------------- |
| `Int` | `2`     | Minimum: `0`, Maximum: `5` |

---

### thresholdPercentage

Threshold Percentage

| Type     | Default | Details                          |
| :------- | :------ | :------------------------------- |
| `Double` | `20.0`  | Minimum: `0.0`, Maximum: `100.0` |

---

### thresholdTick

Threshold Tick

| Type  | Default | Details                      |
| :---- | :------ | :--------------------------- |
| `Int` | `5`     | Minimum: `1`, Maximum: `100` |

---
