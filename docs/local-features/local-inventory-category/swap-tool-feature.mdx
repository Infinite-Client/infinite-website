# SwapToolFeature

Automatically switches to the best tool for the block you are breaking.

## Properties

### method

Method

| Type            | Default  | Details                   |
| :-------------- | :------- | :------------------------ |
| `EnumSelection` | `HotBar` | Options: `Swap`, `HotBar` |

---

### fineToolStrategy

Fine Tool Strategy

| Type            | Default     | Details                                |
| :-------------- | :---------- | :------------------------------------- |
| `EnumSelection` | `SharpTool` | Options: `Shears`, `SharpTool`, `Hand` |

---

### switchDelay

Switch Delay

| Type  | Default | Details                     |
| :---- | :------ | :-------------------------- |
| `Int` | `5`     | Minimum: `0`, Maximum: `20` |
