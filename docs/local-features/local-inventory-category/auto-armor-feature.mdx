# AutoArmorFeature

Automatically equips the best armor in your inventory. Can also switch to Elytra based on durability.

## Properties

### autoEquip

Auto Equip

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### elytraSwitch

Elytra Switch

| Type      | Default | Details |
| :-------- | :------ | :------ |
| `Boolean` | `true`  | -       |

---

### durabilityThreshold

Durability Threshold

| Type  | Default | Details                      |
| :---- | :------ | :--------------------------- |
| `Int` | `5`     | Minimum: `0`, Maximum: `100` |

---

### delay

Equip Delay

| Type  | Default | Details                     |
| :---- | :------ | :-------------------------- |
| `Int` | `2`     | Minimum: `1`, Maximum: `20` |

---
