# ItemRestockFeature

Automatically refills item stacks in your hotbar from your inventory.

## Properties

### delayProperty

Restock Delay

| Type  | Default | Details                     |
| :---- | :------ | :-------------------------- |
| `Int` | `3`     | Minimum: `1`, Maximum: `20` |

---

### thresholdProperty

Restock Threshold

| Type  | Default | Details                     |
| :---- | :------ | :-------------------------- |
| `Int` | `8`     | Minimum: `1`, Maximum: `64` |

---
