# FastBreak
Increases block breaking speed.

## Settings

### SafeMode
Enables safe mode for FastBreak.

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `true`
* **Current Value**: `true`

---

### ThresholdPercentage
Sets the threshold percentage for FastBreak.

#### Setting Information

* **Type**: `Double`
* **Default Value**: `20.0`
* **Current Value**: `20.0`
* **Minimum Value**: `0.0`
* **Maximum Value**: `100.0`

---

### ThresholdPercentage
Sets the threshold percentage for FastBreak.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `5`
* **Current Value**: `5`
* **Minimum Value**: `1`
* **Maximum Value**: `100`

---

### Interval
Sets the execution interval for FastBreak.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `0`
* **Current Value**: `0`
* **Minimum Value**: `0`
* **Maximum Value**: `5`
