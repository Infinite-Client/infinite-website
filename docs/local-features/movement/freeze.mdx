# Freeze
Stops sending movement packets to the server, allowing for teleportation.

## Settings

### Duration
Maximum duration (seconds) before freeze automatically disables. 0 for unlimited.

#### Setting Information

* **Type**: `Float`
* **Default Value**: `0.0`
* **Current Value**: `0.0`
* **Minimum Value**: `0.0`
* **Maximum Value**: `600.0`

---

### PacketLimit
Maximum number of packets to accumulate before freeze automatically restarts. 0 for unlimited.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `0`
* **Current Value**: `0`
* **Minimum Value**: `0`
* **Maximum Value**: `500`
