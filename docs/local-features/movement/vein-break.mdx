# VeinBreak
Destroys entire veins at once.

## Settings

### Range
Sets the range for VeinBreak.

#### Setting Information

* **Type**: `Double`
* **Default Value**: `5.0`
* **Current Value**: `5.0`
* **Minimum Value**: `1.0`
* **Maximum Value**: `6.0`
