# HighStep

Allows stepping up blocks of a specified height.

## Settings

### MaxStepHeight

Maximum height the player can step up.

#### Setting Information

- **Type**: `Double`
- **Default Value**: `1.0`
- **Current Value**: `1.0`
- **Minimum Value**: `1.0`
- **Maximum Value**: `10.0`
