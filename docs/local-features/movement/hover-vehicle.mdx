# HoverVehicle

Makes vehicles act like hover vehicles, allowing faster travel over water.

## Settings

### MaxSpeed

Sets the speed multiplier for the hover vehicle.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `20.0`
- **Current Value**: `20.0`
- **Minimum Value**: `1.0`
- **Maximum Value**: `200.0`

---

### Acceleration

Sets the acceleration power for the hover vehicle.

#### Setting Information

- **Type**: `Float`
- **Default Value**: `0.5`
- **Current Value**: `0.5`
- **Minimum Value**: `0.05`
- **Maximum Value**: `2.0`
