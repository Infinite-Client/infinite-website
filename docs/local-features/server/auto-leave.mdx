# AutoLeave

Automatically leaves the server when certain conditions are met.

## Settings

### HpThreshold

HP percentage below which the server will be left.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `6`
- **Current Value**: `6`
- **Minimum Value**: `0`
- **Maximum Value**: `20`

---

### TotemThreshold

Number of totems remaining to trigger auto-leave.

#### Setting Information

- **Type**: `Int`
- **Default Value**: `0`
- **Current Value**: `0`
- **Minimum Value**: `0`
- **Maximum Value**: `11`

---

### Method

Method used to leave the server.

#### Setting Information

- **Type**: `Enum`
- **Default Value**: `Normal`
- **Current Value**: `Normal`
- **Options**: `Normal`, `InvalidChat`, `SelfHurt`
