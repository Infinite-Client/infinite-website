# AntiCheat
Prevents anti-cheat from detecting certain actions.

## Settings

### EnableForQuickMove
Enables anti-cheat for quick movement.

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `false`
* **Current Value**: `false`

---

### QuickMoveTolerance
Sets the tolerance for quick movement anti-cheat.

#### Setting Information

* **Type**: `Double`
* **Default Value**: `0.05`
* **Current Value**: `0.05`
* **Minimum Value**: `0.0`
* **Maximum Value**: `0.3`

---

### QuickMoveLatencyMultiplier
Sets the latency multiplier for quick movement anti-cheat.

#### Setting Information

* **Type**: `Double`
* **Default Value**: `0.5`
* **Current Value**: `0.5`
* **Minimum Value**: `0.0`
* **Maximum Value**: `1.0`

---
