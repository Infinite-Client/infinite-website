# BranchMiner
Automatically performs branch mining.

## Settings

### BranchLength
Sets the length of the branches.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `32`
* **Current Value**: `32`
* **Minimum Value**: `8`
* **Maximum Value**: `128`

---

### BranchInterval
Sets the interval between branches.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `3`
* **Current Value**: `3`
* **Minimum Value**: `1`
* **Maximum Value**: `5`

---

### MinEmptySlots
Sets the minimum number of empty slots.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `5`
* **Current Value**: `5`
* **Minimum Value**: `1`
* **Maximum Value**: `27`

---

### DetectSpaceThreshold
Sets the space detection threshold.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `64`
* **Current Value**: `64`
* **Minimum Value**: `32`
* **Maximum Value**: `128`

---

### ChestSearchRadius
Sets the chest search radius.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `16`
* **Current Value**: `16`
* **Minimum Value**: `8`
* **Maximum Value**: `64`

---

### ItemCollectionWaitTicks
Sets the item collection wait ticks.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `40`
* **Current Value**: `40`
* **Minimum Value**: `20`
* **Maximum Value**: `100`

---

### EnableTorchPlacement
Enables or disables torch placement during branch mining.

#### Setting Information

* **Type**: `Boolean`
* **Default Value**: `false`
* **Current Value**: `false`

---

### TorchInterval
Sets the interval (in blocks) for placing torches during branch mining.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `8`
* **Current Value**: `8`
* **Minimum Value**: `4`
* **Maximum Value**: `16`
