# ShieldMachine
Automatically builds a shield machine.

## Settings

### Width
Sets the width of the shield machine.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `3`
* **Current Value**: `3`
* **Minimum Value**: `1`
* **Maximum Value**: `6`

---

### Height
Sets the height of the shield machine.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `3`
* **Current Value**: `3`
* **Minimum Value**: `2`
* **Maximum Value**: `5`

---

### Length
Sets the length of the shield machine.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `100`
* **Current Value**: `100`
* **Minimum Value**: `0`
* **Maximum Value**: `256`

---

#### Setting Information

* **Default Value**: `2`
* **Current Value**: `2`
* **Minimum Value**: `1`
* **Maximum Value**: `5`

* **Default Value**: `true`
