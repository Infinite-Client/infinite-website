# WoodMiner
Automatically mines wood.

## Settings

### SearchRadius
Sets the search radius for wood mining.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `16`
* **Minimum Value**: `8`
* **Maximum Value**: `64`

---

### SearchHeight
Sets the search height for wood mining.

#### Setting Information

* **Type**: `Int`
* **Default Value**: `8`
* **Minimum Value**: `4`
* **Maximum Value**: `32`

---

### WoodTypes
List of wood types to mine.

#### Setting Information

* **Type**: `BlockList`
* **Default Value**: `[]`
* **Item Count**: 0
* **List Type**: Block IDs
